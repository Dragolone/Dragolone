## 💫 About Me

I'm a Computer Science undergraduate at **The University of Sydney**, majoring in Software Development and building real-world full-stack products from scratch.

Currently, I'm the sole engineer on a **remote-controllable robotics product**, where I work across the full stack:

- Cross-platform WeChat Mini Program using **Vue 3 / uni-app**
- Backend and cloud services with **uniCloud**
- IoT gateway using **Flask, MQTT, WebSocket**
- Real-time robot telemetry, command delivery, binding logic, and user authentication

I enjoy building products from **0 → 1**, especially when the system involves both software and hardware.  
My focus is not just making things work, but making them reliable, measurable, and maintainable.

Recently, I'm also exploring **LLM application development**, including prompt engineering, function calling, RAG, and agentic workflows.

> I like building practical systems that connect frontend, backend, data, and real-world devices.

- 🤖 Building a remote robot control platform with WeChat Mini Program + MQTT
- ⚡ Improving real-time command delivery and telemetry reliability
- 🧠 Learning LLM application development, RAG, and AI agent workflows
- 🛠️ Turning project ideas into usable full-stack products
